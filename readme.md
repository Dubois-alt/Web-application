# 🌐 HTML Basics Project

## 📖 Description
This project demonstrates the basics of **HTML (HyperText Markup Language)**, the standard language used to create web pages. It includes simple examples that show how to structure content using HTML elements.

---

## 🧠 What is HTML?
HTML is used to define the structure of web pages. It uses elements (tags) to organize content such as headings, paragraphs, links, and images.

---

## 🏗️ Project Structure
The project contains:

- `index.html` – Main webpage file
- `images/` – Folder for images (if any)

---

## ✨ Features
- Basic HTML page structure  
- Use of headings and paragraphs  
- Adding links and images  
- Simple lists (ordered and unordered)  

---

## 🧾 Example Code

```html
<!DOCTYPE html>
<html>
<head>
    <title>My First HTML Page</title>
</head>
<body>
    <h1>Welcome</h1>
    <p>This is a simple HTML page.</p>
</body>
</html>
